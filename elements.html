Nama Elements Attribute yang biasanya dipakai Kegunaan
dd class, id, dll berisi data yang menjelaskan istilah definisi
dt class, id, dll digunakan ketika ingin memberikan nama dalam daftar definisi
dl class, id, dll digunakan ketika ingin memasukan definisi istilah dalam dokumen
sup class, id, dll untuk mengalihkan teks menjadi agak keatas
sub class, id, dll untuk mengalihkan teks menjadi agak kebawah
b, dan strong class, id, dll untuk menebalkan teks atau bold
i class, id, dll untuk memiringkan teks atau bisa disebut italic
a href, dll untuk membuat sebuah text yang berisi link tujuan link
script src, dll untuk membuat script agar bisa diatur dengan file javascript
nav class, id, dll biasanya untuk membuat sebuah navbar
button class, id, onclick, onmouseover, onmouseout, dll untuk membuat sebuah tombol
link rel, href, dll biasanya untuk memanggil sebuah link tujuan untuk manambah sesuatu pada website
meta charset, name, content, dll berfungsi sebagai pemberi informasi metadata dari sebuah halaman web
blackqoute cite, dll digunakan untuk menunjukkan sebuah konten yang dikutip dari sumber lain
