Nama Style Kegunaan
background-(color, image) untuk mengatur warna latar belakang, dan gambar latar belakang
border-(left, right, bottom, top) untuk memunculkan border pada elemen
margin-(left, right, bottom, top) untuk mengatur margin pada elemen
width untuk mengatur panjang ruang pada elemen
height untuk mengatur tinggi ruang pada elemen
transition untuk mengatur transisi pada elemen
.namafile:hover{background-color: yellow;} untuk mengatur perubahan ketika kursor kita menyentuh pada class atau id tujuan
@media screen and (max width: 854px){img{width: 100%;}} untuk menyesuaikan ukuran-ukuran elemen ketika layar sudah melewati batas panjang yang kita atur
box-shadow untuk mengatur sebuah elemen agar bisa mampunyai bayangan
position untuk mengatur posisi elemen
display menentukan bagaimana elemen HTML harus ditampilkan di halaman web
justify-content mengatur sejauh mana item fleksibel menyebar di sepanjang garis utama sebuah elemen flex-container
